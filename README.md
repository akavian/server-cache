# Project Description
Creating a microservice for the porpuse of caching json based documents leveraging MongoDB as the database.
Also, another layer of caching such as in memory caching exists to reduce the calls to the database.
This is intented as a Practice Project to utilize different functionalites of Spring as well as performing other practices such as migrating from Spring Boot 3 to Spring Boot 4.
In addition, dockerizing the project for easy local setup by utilzing a docker image and a docker compose file.
